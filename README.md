## Cómo publicarlo

1. Crea un repositorio público en GitHub.
2. Sube `index.html` y la carpeta `img`.
3. Ve a `Settings > Pages`.
4. Selecciona `Deploy from a branch`.
5. Branch: `main`.
6. Folder: `/ root`.
7. Guarda y comparte la URL que te dé GitHub Pages.
